# Codex + n8n Local Setup

This repository is a small public reference for getting Codex working with a local `n8n` instance on Windows.

## What This Repo Contains

- [1. Local Setup](./1.%20local%20setup.md)
- [2. Upgrading](./2.%20upgrading.md)
- [3. Tunneling Guide](./3.%20tunneling%20guide.md)
- [4. VPS Hosting](./4.%20vps%20hosting.md)

## Recommended Starting Point

Start with the local setup guide first.

Use the upgrading guide when you want to refresh or update your existing setup later.

Move to the tunneling guide when something outside your PC needs to call your local `n8n` during development, such as a webhook or callback.

Use the VPS hosting guide when you want a real always-on public n8n with a stable domain and proper hosting instead of a temporary tunnel.
